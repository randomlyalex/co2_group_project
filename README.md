## How to run
```zsh
cd client
```
### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```
Go to localhost:8080 to view the app in the browser

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
